心里的疑问，我们提前答了
装修前最想搞清楚的事，一句一句回
💭 你们到底是做什么的？
我们是博洛尼和不厌空间联合推出的智能整装套餐，29.8万，三居室全包。博洛尼定制、卡萨帝厨电、全屋智能、施工设计，一站式搞定，拎包入住。
💭 29.8万到底含了什么？
含六大板块：第一，不厌空间全案设计，设计师全程跟；第二，施工40项，防水质保5年；第三，主材全是设计师严选的一线品牌；第四，博洛尼Y系列全屋定制，橱柜衣柜木门全含；第五，卡萨帝烟机灶具；第六，全屋智能，电动窗帘、智能灯光、安防，用手机就能控制。

而且我们是按套内面积计价，不含公摊，等于比别家按建筑面积算便宜了一截。
💭 跟传统装修比到底好在哪？
传统装修你得分别找设计公司、施工队、定制店、智能公司，每个都要对接、都要加钱，最后超预算是常态。我们是一口价29.8万，合同写明无主动增项，一个团队从头跟到尾。

而且我们的定制是博洛尼，不是小品牌；智能是全屋智能系统，不是装个智能音箱就完了；辅料用的是德国爱格板、可耐福、都芳、霍尼韦尔，这些你单独去买，光辅料就不止这个价。
💭 水电保修怎么算？
电路——全改终身质保，断点改造仅保更改部分。老线路不在范围内，这一点我们写清楚，不画大饼。

水管——保50年，前提是工艺达标：主管道换到管道井、配件全用同品牌。新旧管接头厂家不承保，我们不会含糊地承诺"全包"。

简单说：做到的标准保，做不到的坦诚说。
💭 家里有小孩，环保能放心吗？
这是我们最重视的部分。博洛尼板材 ENF级 + JIS F☆☆☆☆ 双认证，比国标E1还高一档；封边用的是激光封边，不用胶水，直接杜绝甲醛释放源；9道检测 + 孔槽全封堵，从工艺上堵死甲醛释放路径；施工用的德国爱格板、都芳墙漆、东方雨虹防水，全是进口或一线环保品牌，清华大学建筑环境检测中心认证。

而且博洛尼定制板材可以签订环保承诺书，ENF级 + JIS F☆☆☆☆ 双认证，甲醛优于国标，不达标免单，白纸黑字写进合同。您有小孩，这点特别重要。
💭 如果我只装一部分呢？
套餐是整套设计的，分开做会破坏整体性。如果预算有限，可以先做基础和定制，智能部分后面再加。
💭 定制风格不喜欢怎么办？
标配是Y系列，想升级也没问题。设计方案定稿后，您挑好喜欢的款式，补个差价就行。
💭 智能设备坏了没人管怎么办？
所有设备有官方保修，我们负责对接售后。以后新买的 iOS 生态设备也能直接接入，不用重新布线。
💭 施工质量靠不靠谱？
按博洛尼钛马赫精工体系执行，40项标配施工，每个关键节点都要您签字确认才能进入下一道工序。防水质保5年。
💭 装完多久才能住进去？
签约后98个工作日交付。ENF级板材+德国爱格板+都芳墙漆，环保达标，完工7天后检测甲醛合格就能入住。
💭 29.8万真的不能再少了？
这个价格已经是套餐价，含了博洛尼定制（市场价11.4万）。确定要签的话可以申请送一个智能门锁，再优惠就没有了。
💭 你们跟别家比到底好在哪？
两个核心优势：第一，博洛尼定制是标配，别的公司要单独买；第二，全屋智能含在套餐内，别的公司要后期自己找人装。一站式搞定。
💭 面积怎么算的？会不会被坑？
